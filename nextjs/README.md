# Next.js Rules

Rules for Next.js 14+ projects using the App Router.

## Files

| File | Purpose | Use When |
|------|---------|----------|
| `project-init-rules.md` | Scaffold a new project | Starting a new Next.js project |
| `development-rules.md` | Core coding conventions | Every session on a Next.js project |
| `feature-rules.md` | Build a new feature end-to-end | Adding a new page or feature |
| `db-rules.md` | Database schema and queries | Working with Prisma or Supabase |
| `api-rules.md` | API route conventions | Creating or modifying API routes |
| `ui-rules.md` | UI component patterns | Building UI in Next.js |

## Recommended Combination

For a full Next.js project session:

```
development-rules.md + api-rules.md + db-rules.md
```
